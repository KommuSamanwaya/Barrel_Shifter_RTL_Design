# Barrel_Shifter_RTL_Design
Verilog RTL implementation of an 8-bit Barrel Shifter supporting Logical Shift Left (LSL), Logical Shift Right (LSR), Rotate Left (ROL), and Rotate Right (ROR), along with functional verification, synthesis, and gate-level schematic generation.
Features:
Logical Shift Left (LSL)
Logical Shift Right (LSR)
Rotate Left (ROL)
Rotate Right (ROR)
Combinational RTL Design
Synthesizable Verilog Code
Functional Verification using Testbench
Synthesized Gate-Level Schematic
